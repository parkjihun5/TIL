## 스프링 부트 개발환경 3가지 준비과정
1. JDK
2. IDE
3. Spring Boot 프로젝트

### JDK 설치
https://www.oracle.com/kr/java/technologies/downloads/

### JDK 설치 확인
```
java -version
```

### 인텔리제이 설치
https://www.jetbrains.com/idea/download/?section=windows

### 프로젝트 만들기
https://start.spring.io/

project = Gradie Project  
Language = Java  
Packaging = Jar  
Java = 본인의 JDK version에 맞게 선택  

Artifact = 프로젝트명  
Dependencies = 도구들  
ex)
- Spring Web
- H2 Database
- Mustache
- Spring Data JPA

설정 및 도구 추가 후 GENERATE 클릭하여 다운로드
